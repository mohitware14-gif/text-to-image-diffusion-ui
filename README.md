# text-to-image-diffusion-ui
AI project using diffusion models to generate images from text prompts with an interactive UI in Google Colab
## 📌 Overview
This project demonstrates how to generate images from text prompts using a pre-trained diffusion model. A simple UI is created using ipywidgets in Google Colab.

## 🚀 Features
- Text-to-image generation
- Negative prompt support
- Adjustable image size
- Interactive UI

## 🛠️ Tools Used
- Python
- Google Colab
- Diffusers Library
- PyTorch

## 🧾 Input Prompt
A futuristic Mumbai city powered by AI, flying cars, neon lights, smart infrastructure, humans and robots coexisting, ultra realistic, cinematic lighting, 4k, highly detailed
## 📓 Negative Prompt
ultra realistic, 4k, cinematic lighting, highly detailed, sharp focus
